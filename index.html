Atividades de Lógica de Programação

Atividade 1 - Algoritmo do Sanduíche 🥪

Objetivo: Entender que um algoritmo é uma sequência de passos bem definidos.

Instruções: Imagine que você vai ensinar um robô a fazer um sanduíche de presunto e queijo. Escreva um passo a passo bem detalhado, como se o robô não soubesse nada.

Atividade 2 - Siga a Sequência 🔢

Objetivo: Identificar padrões e criar sequências lógicas.

Observe as sequências abaixo e tente descobrir o próximo número ou padrão:

  • 2, 4, 6, 8, __ ?
  • 1, 1, 2, 3, 5, 8, __ ?
  • 10, 20, 40, 80, __ ?

Atividade 3 - Fluxograma do Dia 📍

Objetivo: Representar processos de forma visual.

Desenhe um fluxograma simples para representar seu dia desde que acorda até dormir.

Atividade 4 - Jogo do Se e Então 🤔

Objetivo: Introduzir o conceito de condições (estrutura "se... então...").

Complete as frases abaixo usando "se" e "então" para criar regras lógicas:

  • Se estiver chovendo, então ___________________.
  • Se a loja estiver fechada, então ___________________.
  • Se o semáforo estiver vermelho, então ___________________.

Atividade 5 - Histórias Condicionais 📖

Objetivo: Criar histórias usando a lógica condicional.

Complete as histórias a seguir usando "se... então...".

Atividade 6 - Missão Resgate 🚀

Objetivo: Criar regras para um jogo de resgate espacial usando "se... então...".

Defina três regras para um astronauta que precisa resgatar um colega perdido no espaço.
